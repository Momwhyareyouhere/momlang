Installation

1. Add the security keyring:
sudo wget -O /usr/share/keyrings/momlang-archive-keyring.gpg https://momwhyareyouhere.github.io/momlang/momlang-archive-keyring.gpg
2. Add the official repository:
echo "deb [signed-by=/usr/share/keyrings/momlang-archive-keyring.gpg] https://momwhyareyouhere.github.io/momlang/ noble main" | sudo tee /etc/apt/sources.list.d/momlang.list
3. Install the compiler:
sudo apt update && sudo apt install momlang

Upgrade to Newest Version

If a new version is released, run these commands to force an update and bypass the local cache:

sudo apt update -o Acquire::No-Cache=true
sudo apt upgrade momlang

Language Syntax

MomLang uses keywords to manipulate a single global variable called cookies.

KeywordAction
REMEMBER [num]Sets cookies to [num]
GATHER [num]Adds [num] to cookies
DISCARD [num]Subtracts [num] from cookies
show("text" c)Prints text and the current cookie count

Example Script (test.mom)

REMEMBER 10
GATHER 5
DISCARD 2
show("I have " c " cookies left")
