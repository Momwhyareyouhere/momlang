This is the official APT repository for MomLang, an x86_64 Assembly language for pantry management.

Installation

To install MomLang on Ubuntu 24.04 (Noble), run the following commands:

curl -fsSL https://momwhyareyouhere.github.io/momlang/momlang.gpg | sudo gpg --dearmor -o /usr/share/keyrings/momlang-archive-keyring.gpg

echo "deb [signed-by=/usr/share/keyrings/momlang-archive-keyring.gpg] https://momwhyareyouhere.github.io/momlang/ noble main" | sudo tee /etc/apt/sources.list.d/momlang.list

sudo apt update && sudo apt install momlang
